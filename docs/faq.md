# FAQ

The **Frequently Asked Questions (FAQ)** section provides quick answers to common doubts about Xolmis Desktop. It is designed to save time and guide users through recurring issues.

## General

- **What is Xolmis?**  
  Xolmis is an ornithological data management system that helps researchers organize, analyze, and share bird-related data.

- **Is Xolmis free to use?**  
  Yes, Xolmis Desktop is free for research and academic purposes. Check the [License](license.md) for details.

- **Does Xolmis work offline?**  
  Yes, all data is stored locally. Internet is only required for updates or integrations (e.g., eBird).

## Installation and setup

- **Which operating systems are supported?**  
  Xolmis Desktop runs on Windows. Other platforms may be supported in the future.

- **Can I migrate data from another system?**  
  Yes, you can use the [Importing data](importing-data.md) feature to bring external datasets into Xolmis.

## Support

- **Where can I report a bug or request a feature?**  
  Use the [Feedback](feedback.md) page to share suggestions or report issues.
