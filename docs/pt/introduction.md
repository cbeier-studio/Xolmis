# Introdução

## Por que Xolmis?

Dados ornitológicos são frequentemente coletados em campo sob condições desafiadoras e inicialmente registrados manualmente em papel.Posteriormente, esses registros são digitalizados para análise, mas mesmo assim costumam ser armazenados em formatos distintos, como arquivos CSV, planilhas Excel ou documentos de texto, cada um com sua própria estrutura e convenções definidas por diferentes pesquisadores.

Como resultado, os dados podem ficar dispersos em várias pastas, dispositivos ou até mesmo nos computadores de colaboradores. Localizar informações específicas ou lembrar o que existe pode se tornar um verdadeiro desafio. Reutilizar dados de diferentes coleções ou projetos geralmente exige trabalhosos processos de **padronização, reformatação e limpeza**. Embora existam plataformas de ciência colaborativa, uma parte significativa de dados valiosos permanece isolada em máquinas pessoais, inacessível à comunidade científica mais ampla.

O **Xolmis** foi criado para enfrentar essa fragmentação. Ele oferece uma solução unificada para **coletar, organizar, curar e gerenciar dados ornitológicos de diversos tipos**. Projetado para pesquisadores, técnicos de campo, instituições e projetos de conservação, o Xolmis fornece um ambiente centralizado e estruturado que promove:

- **Consistência dos dados** – formatos padronizados entre projetos.  
- **Rastreabilidade** – cada alteração é registrada e vinculada a um usuário.  
- **Reutilização** – conjuntos de dados podem ser compartilhados, exportados e integrados a outras plataformas.  
- **Colaboração** – múltiplos usuários podem trabalhar no mesmo banco de dados com papéis e permissões definidos.  

## O que é Xolmis?

Xolmis é um **sistema integrado para coleta e gerenciamento de dados ornitológicos**. Ele consolida conjuntos de dados dispersos (planilhas, notas de campo e registros digitais) em um banco de dados coerente, pesquisável e seguro.

Seus principais objetivos incluem:

- **Agregação de diversos tipos de dados** oriundos de pesquisas e observações de campo (capturas, avistamentos, ninhos, espécimes, permissões, projetos etc.).  
- **Facilitação da curadoria e validação dos dados**, garantindo precisão e confiabilidade.  
- **Referenciamento cruzado de registros** entre módulos para insights ecológicos e biológicos mais ricos.  
- **Compartilhamento e preservação de longo prazo dos dados**, reduzindo o risco de perda de informações.  
- **Maior consistência, rastreabilidade e segurança** nos fluxos de trabalho da pesquisa ornitológica.  

!!! info
    _Xolmis_ é um gênero de pássaros neotropicais. Atualmente é representado por duas espécies: [_Xolmis irupero_](https://ebird.org/species/whimon1) e [_Xolmis velatus_](https://ebird.org/species/whrmon2). O nome foi escolhido para simbolizar o foco do sistema em ornitologia e sua conexão com a biodiversidade neotropical.

## Aplicações

O Xolmis é composto por duas aplicações distintas, que podem ser usadas de forma independente ou em conjunto, dependendo do fluxo de trabalho:

- [**Xolmis Mobile**](xolmis-mobile.md):  
  Um companheiro de campo leve projetado para **coleta de dados _in situ_**. Permite que pesquisadores registrem capturas, avistamentos e outras observações diretamente em dispositivos móveis, exportando-os posteriormente em formatos padronizados (CSV ou JSON).

- **Xolmis Desktop**:  
  Uma aplicação completa para **gerenciamento de conjuntos de dados complexos**. Oferece ferramentas avançadas para edição, verificação, importação, exportação e análise de dados ornitológicos, além de integração com plataformas externas como o eBird.

Juntas, essas aplicações criam um fluxo de trabalho contínuo: **coletar em campo → consolidar no banco de dados → analisar e compartilhar resultados**.

## Boas práticas ao começar com Xolmis

- **Centralize seus dados**: importe planilhas e notas dispersas em um único banco de dados.  
- **Defina papéis de usuário**: atribua permissões a colaboradores para garantir responsabilidade.  
- **Use módulos padronizados**: registre dados no módulo apropriado (ex.: Capturas, Ninhos, Avistamentos) para manter consistência.  
- **Aproveite a integração entre Mobile e Desktop**: colete dados em campo com o Xolmis Mobile e gerencie-os depois com o Xolmis Desktop.  
- **Planeje a preservação de longo prazo**: configure backups e exporte conjuntos de dados regularmente para evitar perdas.  

Ao adotar o Xolmis, pesquisadores e instituições ganham uma **plataforma robusta, escalável e colaborativa** para o gerenciamento de dados ornitológicos, garantindo que informações valiosas sejam preservadas, acessíveis e prontas para apoiar a conservação e a descoberta científica.
