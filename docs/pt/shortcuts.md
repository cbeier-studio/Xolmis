# Atalhos de teclado

O Xolmis inclui um conjunto de **atalhos de teclado** que tornam a navegação e a entrada de dados mais rápidas e eficientes. O uso de atalhos reduz a dependência do mouse, acelera tarefas repetitivas e ajuda a manter o foco durante a digitação de dados de campo ou o gerenciamento de grandes conjuntos de dados.

## Atalhos globais

Esses atalhos estão disponíveis em toda a aplicação, independentemente do módulo em que você esteja trabalhando:

| Atalho | Função |
| --- | --- |
| ++ctrl+alt+o++ | Conectar a um banco de dados |
| ++ctrl+alt+s++ | Abrir configurações |
| ++ctrl+period++ | Alternar para a próxima aba aberta |
| ++ctrl+comma++ | Alternar para a aba anterior |
| ++f1++ | Abrir ajuda contextual |
| ++ctrl+i++ | Abrir assistente de importação |
| ++ctrl+m++ | Abrir diálogo de importação do Xolmis Mobile |
| | Importar de arquivo de registros do eBird |
| | Importar coordenadas geográficas |

## Atalhos de módulo

Esses atalhos se aplicam a ações dentro dos módulos (ex.: Capturas, Avistamentos, Ninhos, Projetos):

| Atalho | Função |
| --- | --- |
| ++f5++ | Atualizar registros |
| ++ctrl+n++ | Criar um novo registro |
| ++ctrl+q++ | Abrir Entrada rápida |
| ++ctrl+b++ | Criar novo lote de registros (se disponível) |
| ++ctrl+e++ | Editar o registro selecionado |
| ++ctrl+del++ | Excluir o registro selecionado |
| ++ctrl+f++ | Localizar/pesquisar registros |
| ++ctrl+shift+f++ | Opções avançadas de filtragem |
| ++esc++ | Limpar pesquisa |
| ++ctrl+alt+p++ | Abrir visualização de impressão |
| ++ctrl+alt+e++ | Exportar grade ou relatório atual |
| | Exportar coordenadas do registro selecionado (se disponível) |
| ++ctrl+alt+m++ | Abrir visualização de mapa (se disponível) |
| ++ctrl+alt+l++ | Abrir painel de resumo (se disponível) |
| ++ctrl+alt+i++ | Abrir painel lateral de imagens (se disponível) |
| ++ctrl+alt+a++ | Abrir painel lateral de gravações de áudio (se disponível) |
| ++ctrl+alt+v++ | Abrir painel lateral de vídeos (se disponível) |
| ++ctrl+alt+d++ | Abrir painel lateral de documentos e links (se disponível) |
| ++ctrl+alt+r++ | Abrir verificações de registros |
| ++ctrl+j++ | Mostrar/ocultar painel de dados relacionados (se disponível) |
| | Criar um novo registro relacionado |
| | Editar o registro relacionado selecionado |
| | Excluir o registro relacionado selecionado |
| | Ajustar colunas da grade de dados ao conteúdo |
| ++ctrl+equal++ | Aumentar altura das linhas da grade de dados |
| ++ctrl+minus++ | Diminuir altura das linhas da grade de dados |
| ++ctrl+shift+t++ | Transferir anilhas (no módulo Anilhas) |
| ++ctrl+shift+b++ | Abrir balanço de anilhas (no módulo Anilhas) |

## Inserindo e editando dados

Esses atalhos são úteis ao inserir ou editar registros em diálogos:

| Atalho | Função |
| --- | --- |
| ++ctrl+s++ | Salvar o registro atual |
| ++esc++ | Cancelar inserção/edição e fechar o diálogo |
| ++tab++/++enter++ | Ir para o próximo campo |
| ++shift+tab++ | Ir para o campo anterior |

## Conversor de coordenadas

| Atalho | Função |
| --- | --- |
| ++ctrl+o++ | Abrir de arquivo |
| ++ctrl+s++ | Salvar em arquivo |
| ++ctrl+enter++ | Converter coordenadas |

!!! tip
    Habilite a opção **Usar teclas <Enter/Return> para avançar para o próximo campo** em [Configurações](settings.md) para usar ++enter++ como a tecla ++tab++.

## Boas práticas

- **Memorize os atalhos mais comuns**: aqueles usados com frequência durante a entrada de dados.  
- **Use atalhos globais para acesso rápido**: eles economizam tempo ao alternar bancos de dados, ajustar configurações ou navegar entre abas.  
- **Combine com teclas de navegação**: use ++tab++/++enter++ e ++shift+tab++ para mover entre campos durante a edição.  
- **Evite exclusões acidentais**: tenha cuidado com ++ctrl+del++; confirme antes de excluir registros importantes.  
- **Use atalhos de mídia**: abra rapidamente painéis laterais de imagens, áudio ou vídeos sem sair do registro.  
- **Pratique fluxos de trabalho**: incorpore atalhos à sua rotina diária para melhorar a eficiência.  

## Relação com outros módulos

Os atalhos de teclado aumentam a produtividade em todos os módulos:

- **Capturas e Avistamentos** – Adicione ou edite registros rapidamente durante a digitação de dados.  
- **Ninhos e Ovos** – Acelere a entrada repetitiva de dados de reprodução.  
- **Projetos e Permissões** – Simplifique o gerenciamento de registros administrativos.  
- **Instituições e Pesquisadores** – Facilite atualizações de informações de colaboradores.  
- **Módulos Geo** – Acesse mapas e resumos mais rapidamente com atalhos dedicados.  
- **Módulos de mídia** – Anexe e gerencie arquivos multimídia com mais eficiência.  

Ao dominar os atalhos de teclado, os pesquisadores podem otimizar seu fluxo de trabalho no Xolmis, tornando o gerenciamento de dados **mais rápido, consistente e menos propenso a erros**.
