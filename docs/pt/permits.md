# Licenças

Em muitos projetos de pesquisa, **licenças** são uma exigência burocrática recorrente. Uma licença representa uma permissão oficial que permite aos pesquisadores realizar atividades específicas, como anilhamento de aves, coleta de espécimes, levantamentos de campo ou transporte de material biológico. Gerenciar licenças de forma adequada garante conformidade com regulamentos legais e políticas institucionais, além de ajudar a manter a transparência nas atividades de pesquisa.

Abra o módulo Licenças no menu principal: **Gestão → Licenças**.

## Adicionando ou editando uma licença

Ao criar ou editar um registro de licença, os seguintes campos estão disponíveis:

| Campo | Obrigatório | Descrição |
| --- | --- | --- |
| **Nome** | Sim | Nome curto para identificar a licença |
| **Número da licença** |  | Número oficial do documento de licença |
| **Tipo** | Sim | Tipo de licença: Licença de anilhamento, Licença de coleta, Licença de pesquisa, Licença de entrada, Licença de transporte, Outra licença |
| **Emissor** | Sim | Instituição ou organização que emitiu a licença |
| **Data de emissão** | Sim | Data em que a licença foi emitida |
| **Data de validade** |  | Data em que a licença expira (se aplicável) |
| **Notas** |  | Qualquer informação adicional sobre a licença |

## Boas práticas

- **Mantenha as licenças atualizadas**: sempre registre a data de validade e renove antes que expirem.  
- **Relacione licenças a projetos**: associe licenças aos projetos de pesquisa relevantes para garantir rastreabilidade.  
- **Registre instituições emissoras**: documente claramente o emissor para facilitar auditorias e verificações de conformidade.  
- **Use notas para detalhes**: adicione informações como restrições, condições ou escopo geográfico da licença.  
- **Centralize o gerenciamento**: armazene todas as licenças no Xolmis para evitar perda de documentos entre diferentes equipes.  

## Relação com outros módulos

As licenças estão interligadas a outras partes do Xolmis:

- **[Projetos](projects.md)**: licenças são frequentemente necessárias para permitir atividades de projeto.  
- **[Pesquisadores](researchers.md)**: pesquisadores individuais podem precisar de licenças específicas para realizar trabalho de campo.  
- **[Capturas](captures.md) e [Espécimes](specimens.md)**: atividades de anilhamento, coleta de espécimes e amostragem devem estar vinculadas a licenças válidas.  

Ao gerenciar licenças no Xolmis, você garante que todas as atividades de pesquisa estejam legalmente autorizadas, devidamente documentadas e facilmente acessíveis para relatórios ou auditorias.
