# Interface do Usuário

A interface do Xolmis foi projetada para suportar diferentes **experiências de uso**. Em vez de aprender cada botão isoladamente, pense na interface como um conjunto de áreas que orientam você nas principais tarefas: navegar, trabalhar com registros, explorar relacionamentos e acessar ferramentas complementares.

## Navegação

É assim que você se movimenta pelo sistema e encontra o que precisa.

- **Menu principal**: A barra superior organiza os módulos por tema (Arquivo, Amostragem, Indivíduos, Reprodução, Gestão, Geo, Taxonomia, Ajuda).  
- **Barra de navegação**: Logo abaixo do menu, com botões de acesso rápido (*Início* :material-home:, *Novo* :material-plus-circle:), abas para módulos abertos, caixa de busca :octicons-search-16: e notificações :material-bell:.  

Use essas ferramentas para **alternar rapidamente entre módulos**, **abrir novos registros** e **pesquisar em diferentes conjuntos de dados**.

## Trabalhando com registros

É aqui que você passa a maior parte do tempo: visualizando, editando e gerenciando dados.

- **Painel principal de registros**: A grade que exibe registros, semelhante a uma planilha.  
- **Barra de ferramentas de registros**: Ações como *Novo registro* :material-plus-circle:, *Entrada rápida* :material-table-large-plus:, *Editar* :material-pencil:, *Marcar/desmarcar todos os registros* :material-checkbox-multiple-marked:, *Exportar* :material-share:, *Imprimir* :material-printer:, *Limpar todos os filtros* :material-filter-remove:, *Excluir* :material-delete:.  
- **Barra de status dos registros**: Mostra quantos registros correspondem à sua consulta, permite navegar (*Primeiro* :material-page-first:, *Anterior* :material-chevron-left:, *Próximo* :material-chevron-right:, *Último* :material-page-last:), atualizar registros :material-refresh:, abrir [histórico de registros](record-history.md) :material-history: e exibir [verificação de registros](record-verifications.md) :material-checkbox-multiple-marked-circle:.  

Esses controles permitem **adicionar, editar, filtrar e exportar registros** de forma eficiente.

## Explorando relacionamentos

Muitos registros estão conectados — espécimes a levantamentos, ovos a ninhos, capturas a indivíduos.

- **Painel de registros relacionados**: Área expansível na parte inferior da janela.  
- **Barra de navegação**: Abas para alternar entre módulos relacionados.  
- **Barra de ferramentas**: Entrada rápida, adições em lote, adição/edição rápida, exportar, excluir.  
- **Barra de status**: Semelhante à grade principal, mostrando contagem de registros e navegação.  
- **Painel lateral (quando disponível)**: Contexto extra, por exemplo, balanço de orçamento de projeto.  

Esse painel ajuda você a **visualizar conexões** e **gerenciar dados vinculados** sem sair do registro principal.

## Ferramentas complementares

No lado direito, a **barra lateral de ferramentas** abre painéis que enriquecem seu trabalho:

- **Detalhes**: Campos completos do registro.  
- **[Filtros rápidos](search-and-filtering-data.md#filtros-rapidos)**: Aplicar ou limpar filtros.  
- **[Imagens](images.md), [áudio](audio-recordings.md), [vídeo](videos.md)**: Gerenciar arquivos multimídia.  
- **[Documentos e links](documents.md)**: Anexar referências.  
- **[Mapa](map.md)**: Visualizar coordenadas.  
- **[Resumo](summary.md)**: Contagens ou médias rápidas.  
- **Colunas**: Personalizar campos visíveis e layout da grade.  
    - O painel lateral lista todas as colunas disponíveis para a grade do módulo atual.  
    - Cada coluna pode ser alternada entre **visível/invisível** usando as caixas de seleção.  
    - A ordem das colunas na lista define sua ordem na grade.  
    - Ajustar altura das linhas e largura das colunas ajuda a otimizar a exibição dependendo do tipo de dado (ex.: códigos curtos vs. notas longas).  
- **Lixeira**: Gerenciar registros excluídos.  
    - Restaurar selecionado :material-delete-restore: – Restaura o registro selecionado (reativa o registro).  
    - Excluir permanentemente :material-delete-forever: – Remove o registro selecionado do banco de dados de forma definitiva.  

Cada painel possui sua própria mini-barra de ferramentas (ex.: visualizar imagens, limpar filtros, restaurar registros).

## Feedback do sistema

Na parte inferior, a **barra de status principal** mostra:

- Conexão atual do banco de dados  
- Usuário ativo  
- Mensagens temporárias  
- Próximo backup  
- Status da tecla Enter como Tab  
- Status do log do sistema  
- Versão da aplicação  

Essa área mantém você informado sobre seu ambiente e o estado do sistema.
