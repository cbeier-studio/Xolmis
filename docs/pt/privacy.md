# Política de Privacidade do Usuário

Estamos comprometidos em proteger suas informações pessoais. Esta Política de Privacidade explica como tratamos seus dados quando você utiliza nossas aplicações.

## 1. O que coletamos

**O Xolmis não coleta nem armazena quaisquer dados pessoais de usuários.**

Todos os dados ornitológicos inseridos no sistema são **gerenciados localmente pelo usuário** e **nunca são transmitidos para servidores externos** ou compartilhados automaticamente.

## 2. Propriedade e controle dos dados

Você mantém **total propriedade e controle** sobre os dados que insere no Xolmis. O sistema foi projetado para ajudá-lo a organizar e gerenciar seus registros ornitológicos de forma **segura e privada**.

## 3. Compartilhamento de dados

**Nenhum dado é compartilhado com terceiros.**

Qualquer compartilhamento de dados é **inteiramente manual** e fica a **critério do usuário**.

## 4. Dados de dispositivo e uso

O Xolmis não coleta automaticamente informações de uso ou do dispositivo. No entanto, **logs de uso podem ser habilitados opcionalmente**. Esses logs são **armazenados localmente** e podem ser **enviados manualmente ao suporte técnico** para ajudar no diagnóstico de problemas ou na melhoria de funcionalidades.

## 5. Segurança

Embora o Xolmis não transmita dados externamente, recomendamos que os usuários sigam **boas práticas de segurança em seus dispositivos** e realizem **backups regulares dos dados importantes**.

## 6. Alterações nesta política

Esta Política de Privacidade pode ser atualizada no futuro caso o Xolmis passe a coletar ou processar dados de usuários.

**Qualquer alteração será claramente comunicada** dentro do aplicativo ou em nosso site.
