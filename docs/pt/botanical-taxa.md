# Táxons botânicos

O módulo **Táxons botânicos** é usado para registrar e gerenciar táxons de plantas que são relevantes para a pesquisa ornitológica. As plantas desempenham um papel fundamental na ecologia das aves, servindo como substratos de nidificação, fontes de alimento ou componentes do habitat. Ao documentar táxons botânicos, o Xolmis permite que os pesquisadores vinculem espécies de plantas a ninhos, habitats e interações ecológicas.

Abra o módulo Táxons botânicos no menu principal: **Taxonomia → Táxons botânicos**.

## Adicionando ou editando um táxon botânico

Ao criar ou editar um registro de táxon botânico, os seguintes campos estão disponíveis:

| Campo | Obrigatório | Descrição |
| --- | --- | --- |
| **Nome científico** | Sim | Nome científico do táxon (ex.: *Mimosa bimucronata*) |
| **Autoria** |  | Citação da autoria do nome científico (ex.: "Kuntze") |
| **Categoria taxonômica** | Sim | Categoria taxonômica do táxon (espécie, gênero, família etc.) |
| **Nome vernacular** |  | Qualquer nome vernacular ou popular para o táxon (ex.: "maricá") |
| **Táxon pai** |  | Táxon de nível superior ao qual o registro está vinculado (ex.: gênero para uma espécie) |
| **Nome válido** |  | Nome válido selecionado se o táxon for considerado um sinônimo |

!!! tip
    Clique no botão **folha** :material-leaf: no campo **Nome científico** para abrir o Plantminer, uma ferramenta online para verificar se o nome botânico (gênero, espécie ou subespécie) é válido.

## Boas práticas

- **Use nomes científicos aceitos**: sempre siga a nomenclatura botânica reconhecida para garantir consistência.  
- **Inclua a autoria**: ajuda a distinguir entre homônimos e esclarecer a autoridade taxonômica.  
- **Registre nomes vernaculares**: útil para equipes de campo e colaboradores locais que podem não usar nomes científicos.  
- **Vincule a táxons superiores**: mantenha a estrutura hierárquica (espécie → gênero → família → ordem).  
- **Trate sinônimos com cuidado**: use o campo "Nome válido" para evitar duplicações e garantir alinhamento taxonômico adequado.  

## Relação com outros módulos

Os táxons botânicos estão interligados a várias partes do Xolmis:

- **[Ninhos](nests.md)** – plantas são frequentemente usadas como suportes de ninhos; vincular táxons fornece contexto ecológico.  

Ao gerenciar táxons botânicos no Xolmis, os pesquisadores garantem que os dados relacionados às plantas sejam padronizados, rastreáveis e integrados aos estudos ornitológicos.

## Exemplo de fluxo de trabalho

1. Registre uma espécie de planta (*Mimosa bimucronata*) no módulo Táxons botânicos.  
2. Vincule-a como **planta de suporte** em um registro de ninho.  
3. Use nomes vernaculares ("maricá") para facilitar a comunicação com auxiliares de campo.  
4. Gere relatórios que resumam substratos de ninhos por espécie de planta.  

Esse fluxo de trabalho destaca como os táxons botânicos enriquecem os conjuntos de dados ecológicos e fortalecem a conexão entre aves e seus habitats.
