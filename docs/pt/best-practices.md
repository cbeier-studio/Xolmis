# Boas práticas

A seção de **Boas práticas** fornece recomendações para garantir o uso eficiente, consistente e confiável do Xolmis Desktop.

## Entrada de dados

- **Use nomes e códigos claros**: números de campo, abreviações e identificadores devem ser únicos e descritivos.  
- **Registre coordenadas precisas**: sempre verifique a ordem de longitude e latitude.  
- **Vincule registros relacionados**: conecte espécimes a indivíduos, ninhos ou ovos para melhor rastreabilidade.  

## Gerenciamento de dados

- **Evite duplicação**: verifique se um registro já existe antes de criar um novo.  
- **Use categorias de forma consistente**: agrupe métodos, espécimes e registros de maneira lógica.  
- **Documente o contexto**: adicione notas para condições incomuns, adaptações ou circunstâncias especiais.  

## Backup e segurança

- **Agende backups**: configure backups automáticos (diários ou semanais) e armazene-os em um serviço de nuvem.  
- **Verifique os backups**: cheque periodicamente se os arquivos de backup estão completos e acessíveis.  
- **Gerencie permissões**: atribua papéis de usuário com cuidado para proteger dados sensíveis.  

## Eficiência do fluxo de trabalho

- **Aprenda atalhos de teclado**: use atalhos para navegação e entrada de dados mais rápidas.  
- **Personalize grades e filtros**: adapte a interface para destacar as informações mais relevantes.  
- **Use o Quick Entry**: adicione múltiplos registros de forma eficiente durante a digitalização de dados de campo.  

## Colaboração

- **Padronize métodos**: garanta que todos os membros da equipe utilizem os mesmos métodos de amostragem para comparabilidade.  
- **Compartilhe feedback**: incentive os usuários a relatar problemas ou sugerir melhorias.  
- **Integre com plataformas externas**: use o eBird ou outras exportações para compartilhar dados amplamente.  

---

Ao seguir estas boas práticas, os pesquisadores garantem que os dados no Xolmis permaneçam **precisos, consistentes e valiosos** para estudos ornitológicos de longo prazo.
