# Manual do Usuário do Xolmis

Bem-vindo ao **Manual do Usuário do Xolmis Desktop**

Xolmis é um sistema de gerenciamento de dados ornitológicos projetado para ajudar pesquisadores, estudantes e instituições a organizar, analisar e compartilhar dados relacionados a aves com precisão e consistência.

## O que você encontrará aqui

Este manual é seu companheiro para explorar o Xolmis. Ele vai ajudá-lo a:

- **Começar rapidamente** – Aprenda a instalar, configurar e criar seu primeiro banco de dados.  
- **Dominar o essencial** – Descubra ferramentas para adicionar, editar, pesquisar e exportar dados.  
- **Explorar módulos especializados** – De levantamentos e espécimes a ninhos e taxonomia, cada módulo é explicado em detalhe.  
- **Conectar os pontos** – Veja como o Xolmis se integra a plataformas externas como o eBird e dá suporte a coleções científicas.  
- **Manter o controle** – Entenda configurações, privacidade e opções de backup para manter seus dados seguros.  

## Como usar este manual

- Comece com **Primeiros passos** se você é novo no Xolmis.  
- Em seguida, vá para **Usando o Xolmis** para conhecer ferramentas gerais e fluxos de trabalho.  
- Aprofunde-se em **Módulos** para orientações detalhadas sobre tipos específicos de dados.  
- Consulte **Sobre o Xolmis** para informações de contexto, políticas e aspectos legais.  
- Use a **Busca** integrada para encontrar tópicos rapidamente.  

## Uma nota para exploradores

O Xolmis foi desenvolvido para tornar os dados ornitológicos **rastreáveis, padronizados e comparáveis** entre projetos. Seja documentando um único ninho, gerenciando milhares de capturas ou preparando espécimes para um museu, este manual irá guiá-lo passo a passo.
