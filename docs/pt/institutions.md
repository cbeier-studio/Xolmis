# Instituições

O módulo **Instituições** é usado para registrar e gerenciar organizações relevantes para a pesquisa e conservação ornitológica. Uma instituição pode ser uma **universidade, ONG, centro de pesquisa, órgão governamental, empresa ou qualquer outro tipo de entidade** envolvida em projetos, autorizações ou colaborações. Ao documentar instituições, o Xolmis garante que todos os registros estejam devidamente vinculados ao seu contexto organizacional, fortalecendo a responsabilidade e a colaboração.

Abra o módulo Instituições no menu principal: **Gestão → Instituições**.

## Adicionando ou editando uma instituição

Ao criar ou editar um registro de instituição, os seguintes campos estão disponíveis:

| Campo | Obrigatório | Descrição |
| --- | --- | --- |
| **Nome** | Sim | Nome oficial completo da instituição |
| **Abreviação** | Sim | Abreviação ou sigla comum usada para a instituição |
| **Código postal** |  | CEP (Brasil), ZIP Code (EUA) ou equivalente |
| **Endereço** |  | Endereço postal da instituição |
| **Detalhes do endereço** |  | Detalhes complementares como prédio, andar ou número de apartamento |
| **Bairro** |  | Subdivisão ou distrito da cidade |
| **Município** |  | Município, cidade, vila ou equivalente |
| **Estado** |  | Estado ou província do país |
| **País** |  | País onde a instituição está localizada |
| **Pessoa de contato** |  | Nome da pessoa de contato dentro da instituição |
| **E-mail** |  | Endereço de e-mail da instituição ou da pessoa de contato |
| **Telefone** |  | Número de telefone para contato institucional |
| **Notas** |  | Qualquer informação adicional sobre a instituição (ex.: foco de pesquisa, parcerias) |

## Boas práticas

- **Use nomes e abreviações oficiais**: garanta clareza e evite duplicações.  
- **Mantenha as informações de contato atualizadas**: instituições frequentemente mudam e-mails ou telefones; atualize os registros regularmente.  
- **Documente a hierarquia organizacional**: utilize os campos de detalhes de endereço, município e estado para manter dados de localização precisos.  
- **Adicione notas para contexto**: registre informações como áreas de pesquisa, programas de conservação ou colaborações.  

## Relação com outros módulos

As instituições estão interligadas a várias partes do Xolmis:

- **[Pesquisadores](researchers.md)** – Cada pesquisador pode estar afiliado a uma ou mais instituições.  
- **[Membros de projeto](projects.md#membros-do-projeto)** – Projetos são frequentemente coordenados e executados por pesquisadores vinculados a instituições.  
- **[Licenças](permits.md)** – Licenças são emitidas por instituições ou órgãos governamentais.  

Ao gerenciar instituições no Xolmis, os pesquisadores garantem que os dados organizacionais estejam **padronizados, rastreáveis e integrados** aos registros ornitológicos.
