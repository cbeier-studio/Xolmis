# FAQ

A seção de **Perguntas Frequentes (FAQ)** fornece respostas rápidas para dúvidas comuns sobre o Xolmis Desktop. Ela foi projetada para economizar tempo e orientar os usuários em questões recorrentes.

## Geral

- **O que é o Xolmis?**  
  Xolmis é um sistema de gerenciamento de dados ornitológicos que ajuda pesquisadores a organizar, analisar e compartilhar dados relacionados a aves.

- **O Xolmis é gratuito para uso?**  
  Sim, o Xolmis Desktop é gratuito para fins de pesquisa e acadêmicos. Consulte a [Licença](license.md) para mais detalhes.

- **O Xolmis funciona offline?**  
  Sim, todos os dados são armazenados localmente. A internet só é necessária para atualizações ou integrações (por exemplo, eBird).

## Instalação e configuração

- **Quais sistemas operacionais são suportados?**  
  O Xolmis Desktop funciona no Windows. Outras plataformas podem ser suportadas no futuro.

- **Posso migrar dados de outro sistema?**  
  Sim, você pode usar o recurso de [Importação de dados](importing-data.md) para trazer conjuntos de dados externos para o Xolmis.

## Gerenciamento de dados

- **Posso recuperar registros excluídos?**  
  Registros excluídos permanecem inativos por um período configurável (por exemplo, 30–120 dias). Após esse prazo, são removidos permanentemente.

- **Como faço backup dos meus dados?**  
  Use as opções de [Backup e restauração](settings.md#backup-e-restauracao) em Configurações. É recomendado armazenar backups em um serviço de nuvem.

## Suporte

- **Onde posso relatar um erro ou solicitar uma funcionalidade?**  
  Utilize a página de [Feedback](feedback.md) para enviar sugestões ou relatar problemas.

- **O Xolmis coleta dados pessoais?**  
  Nenhum dado pessoal ou sensível é registrado em logs ou compartilhado externamente. Consulte a [Política de Privacidade](privacy.md) para mais detalhes.
