# Xolmis Mobile

**[Xolmis Mobile](https://github.com/cbeier-studio/xolmis_mobile)** é o companheiro de campo do Xolmis Desktop. Ele foi projetado para tornar a coleta de dados ornitológicos mais rápida e fácil durante expedições, levantamentos e observações oportunísticas. Enquanto a versão Desktop foca em **gerenciamento, análise e geração de relatórios de dados**, o aplicativo Mobile é otimizado para **entrada de dados em tempo real no campo**.

!!! note
    Este manual é focado no **Xolmis Desktop**. Para instruções detalhadas sobre o uso do **Xolmis Mobile**, consulte o manual ou a documentação dedicada ao Mobile.

## Principais funcionalidades

- **Entrada rápida de dados** – Registre avistamentos, capturas, ninhos e ovos diretamente em campo.  
- **Modo offline** – Colete dados mesmo sem acesso à internet.  
- **Integração com Desktop** – Importe registros do Mobile para o Desktop para verificação, análise e armazenamento de longo prazo.  
- **Métodos padronizados** – Utiliza os mesmos métodos pré-carregados para garantir compatibilidade.  
- **Suporte a multimídia** – Anexe fotos e notas aos registros durante o trabalho de campo.  

## Fluxo de trabalho com Xolmis Desktop

1. **Prepare seu projeto** no Xolmis Desktop (métodos, pesquisadores, localidades).  
2. **Colete dados** em campo usando o Xolmis Mobile.  
3. **Exporte** os dados do Mobile.  
4. **Importe no Desktop** para verificar, analisar e arquivar os registros.  

Esse fluxo garante que os dados de campo sejam **rastreáveis, padronizados e integrados** ao banco de dados de pesquisa mais amplo.

## Boas práticas

- **Planeje com antecedência**: configure métodos, pesquisadores e localidades no Desktop antes de ir a campo.  
- **Use offline**: não se preocupe com conectividade; o Mobile armazena os dados localmente.  
- **Verifique após importar**: sempre revise os registros do Mobile no Desktop para garantir precisão.  
- **Mantenha os dispositivos carregados**: o trabalho de campo pode ser exigente; assegure que seu dispositivo móvel tenha bateria suficiente.
