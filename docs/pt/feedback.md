# Feedback

Agradecemos seu feedback para ajudar a melhorar o Xolmis! Se você encontrou um erro ou tem uma ideia para uma nova funcionalidade, pode enviar sua contribuição diretamente pelo repositório GitHub na seção **Issues**.

## Como enviar feedback

1. Acesse a [página de Issues do Xolmis no GitHub](https://github.com/cbeier-studio/Xolmis/issues)  
2. Clique em **New Issue**  
3. Escolha um dos modelos disponíveis:  
    - **Bug Report** (Relato de erro)  
    - **Feature Request** (Solicitação de funcionalidade)  
4. Preencha o formulário com o máximo de detalhes possível  
5. Clique em **Submit new issue**  

!!! note

    Para criar issues no GitHub, você precisa estar logado. Se não estiver, ao clicar no botão **New issue** será solicitado que você faça login. Você pode usar uma conta GitHub existente, entrar com sua conta Google ou criar uma nova conta GitHub, se necessário.  

Agradecemos suas contribuições e esperamos tornar o Xolmis ainda melhor com a sua ajuda!
